# tsseed

Basic typescript project seed generator.

Installation:

```sh
npm i -g @lunarade/tsseed
```

Usage:

```sh
tsseed my-project

# use -o to attempt to open the project with vscode
tsseed my-project -o
```

With npx:

```sh
npx @lunarade/tsseed tsseed my-project -o
```
